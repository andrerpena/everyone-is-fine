# North Star — EveryoneIsFine

## Vision

EveryoneIsFine is a colony simulator in the tradition of RimWorld and Dwarf Fortress, built as a web-based game. It is an AI-first game — designed to be programmed by AI agents and completely playable by AI agents.

- The player starts by selecting 3 colonists, each with a unique set of pros and cons.
- The player lands in a deterministically generated map and must build a base and survive.
- A story unfolds, unique to each game session.
